# Transversal · Cadencia  *(el ritmo)*

**Qué es:** el reloj. El ritmo al que la empresa se mira y decide.
**Qué vive aquí:** el pulso de reuniones (diario/semanal/mensual/trimestral/anual). Cada reunión = un latido (cruce de cadencia × sistema nervioso).
**Herramientas:** pulso de reuniones · agenda. Dirigir/decidir = **humano**; convocar, pre-rellenar KPIs y perseguir lo pendiente = **agente**.
