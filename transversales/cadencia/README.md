# Transversal · Cadencia  *(el ritmo)*

> No es un pilar: cruza a los cinco. Es el **reloj** — el ritmo al que la empresa se mira y decide.
> La cadencia es el punto de fallo más común de los sistemas operativos: sin disciplina de ritmo, el sistema se cae.

## Qué vive aquí
El **ritmo de reuniones** (diario, semanal, mensual, trimestral, anual). Cada reunión es un **latido** donde se revisan unos KPIs/hitos y se decide. La reunión es el cruce de **Cadencia** (cuándo) y **Sistema Nervioso** (qué se mira).

## Cómo se relaciona
Cada pilar tiene su ritmo de revisión (el cuál, de cada empresa). La Cadencia pone el **cuándo** a cada KPI y a cada hito.

## El patrón de herramienta (humano vs agente)
| La tarea | La hace |
|---|---|
| Fijar el ritmo y dirigir la reunión / decidir | **Humano** |
| Preparar la reunión (juntar KPIs, agenda, perseguir lo pendiente) | **Agente** |
| Registrar acuerdos y acciones | **Agente** |
