# Transversal · Sistema Nervioso  *(la información)*

> No es un pilar: cruza a los cinco. Es el **cableado** que lleva la información. **Los KPIs nacen aquí.**

## El recorrido del dato
captar el dato (de los procesos) → ordenarlo (**diccionario de datos**) → convertirlo en KPI → detectar (comparar con el umbral) → avisar a quien decide → mostrar el estado (**panel**) → **recordar (el cerebro)**.

## El cerebro — el aprendizaje (dónde vive el "no morir")
La memoria del sistema nervioso no es un cuaderno: es el cerebro que **reconoce** ("esto ya pasó, así se arregló") y por eso el problema no vuelve. El bucle:

> **Un KPI se sale → es un problema → causa raíz → acción correctiva → vuelve al cerebro.**

La acción correctiva cambia lo que haga falta (el proceso, la SOP, el responsable, el plan) y el cerebro lo recuerda para reconocerlo la próxima vez. El aprendizaje continuo es la parte agéntica de este cerebro.

## Cómo se relaciona
Es la única pieza que toca a todas. Cada pilar le entrega sus datos y recibe sus KPIs y sus avisos.

## El patrón de herramienta (humano vs agente)
Captar, ordenar, calcular, detectar, avisar, mostrar, registrar → casi todo **agéntico** una vez el dato es limpio. Lo **humano**: qué significan los números y la decisión.
> Aviso: agentizar **antes** de que el dato esté limpio solo automatiza el desorden.
