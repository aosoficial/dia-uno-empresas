# Transversal · Sistema Nervioso  *(la información)*

**Qué es:** cómo se mueve la información por la empresa. Su sustancia son los datos; **los KPIs nacen aquí**.
**El recorrido del dato:** captar → ordenar (diccionario de datos) → KPI → detectar → avisar a quien decide → mostrar (panel) → **recordar (el cerebro)**.
**El cerebro = el aprendizaje** (el "no morir"): un KPI se sale → problema → causa raíz → acción correctiva → vuelve al cerebro, que lo reconoce la próxima vez.
**Herramientas:** panel de indicadores · diccionario de datos · el cerebro. Casi todo **agéntico** una vez el dato es limpio; lo **humano**: qué significan los números y la decisión.
