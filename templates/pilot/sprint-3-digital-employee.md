# Sprint 3 — Digital Employee

Outcome: one digital employee completes one safe internal task under human review.

## Required setup

- SOUL or identity.
- Memory scope.
- Operations rules.
- Permissions file.
- Forbidden actions.

## Run loop

1. Provide context packet.
2. Ask for draft or analysis only.
3. Human reviews.
4. Record correction or approval.
5. Write Receipt.

## Done when

The digital employee produced useful work, stayed inside permissions and left evidence.
