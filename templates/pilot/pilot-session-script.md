# Pilot Session Script

## Opening

Confirm the scope: this is a guided AI-First operating-system pilot for a service business. The goal is one verified Punto B slice, not a full company transformation in one call.

## Questions

1. What business type are we installing: agency, consultancy or freelancer?
2. What is the current Punto A symptom that hurts revenue, delivery or margin most?
3. Which department owns the bottleneck?
4. What internal task can be run safely without contacting customers or changing production?
5. Who approves external, public, economic, legal, production or sensitive actions?

## Live work

- Run or review the wizard output.
- Fill Direction / Mother Brain.
- Select first department.
- Define first digital employee loop.
- Write the required evidence list.

## Close

Do not close the session until these are explicit:

- next action;
- owner;
- approval boundary;
- evidence created;
- blocker if unresolved;
- Point B readiness command.
