# Operator Checklist

## Before the pilot

- [ ] Confirm company type.
- [ ] Confirm private workspace path.
- [ ] Confirm no secrets or raw customer data are pasted.
- [ ] Confirm human approver.
- [ ] Confirm first bottleneck.

## During the pilot

- [ ] Run wizard.
- [ ] Verify installation.
- [ ] Review approval boundaries.
- [ ] Select one department.
- [ ] Create one context packet.
- [ ] Run one draft-only digital employee task.
- [ ] Human reviews output.

## After the pilot

- [ ] Write Receipt.
- [ ] Write StateChange if operating state changed.
- [ ] Update scorecard.
- [ ] Run Point B validator.
- [ ] Choose next sprint.

## Stop conditions

Stop and ask for Approval before any external, public, economic, legal, production or sensitive action.
