# Generated Company Instance

This folder is the private working copy for one real or demo company.

Do not commit real company data to the public framework repository.

## Start here

1. Read `MAP.md`.
2. Read `AGENTS.md`.
3. Read `company/approval-boundaries.md`.
4. Review `digital-employees/ceo-operations-assistant/`.
5. Save receipts in `receipts/`.

## Structure

- `company/`: company-level operating memory.
- `departments/`: department brains and SOPs.
- `digital-employees/`: employee runtime packs.
- `context-packets/`: context loaded before work.
- `receipts/`: evidence of completed work.
- `statechanges/`: records of operational changes.
- `handoffs/`: session-to-session handoffs.
- `contracts/`: scoped work contracts.
- `traces/`: raw technical/debug references.
- `secrets/`: instructions only; never store real secrets here.
