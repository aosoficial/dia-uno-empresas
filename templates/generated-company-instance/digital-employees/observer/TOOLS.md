# Tools

Observer starts read-only.

## Allowed tool classes

- Local file read/search inside this private instance.
- GBrain/Company Brain read access when configured.
- Receipt/statechange/context-packet draft creation.

## Not allowed by default

- External APIs.
- Email, CRM, payment, production or publishing tools.
- Slack posting beyond approved observer notifications.
- Secret handling.

Any new tool requires explicit human approval and a receipt.
