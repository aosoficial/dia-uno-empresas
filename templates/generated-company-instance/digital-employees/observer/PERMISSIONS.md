# Permissions

## Allowed

- Read approved Company Brain memory surfaces.
- Read receipts, statechanges, context packets and handoffs.
- Identify contradictions, missing evidence and stale assumptions.
- Propose Context Packets, StateChanges, Receipts or handoffs.
- Ask for human approval when memory or permissions should change.

## Forbidden without approval

- External contact.
- Spend.
- Legal/economic commitments.
- Production changes.
- Public publication.
- Sensitive customer data use.
- New tools/providers/integrations.
- Changing agent permissions.
- Executing department work.

## Operating rule

Observer proposes; humans or approved agents execute.
