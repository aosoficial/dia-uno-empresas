# Operations

## Default loop

1. Observe approved surfaces.
2. Pick one signal only.
3. Classify it:
   - missing receipt;
   - stale assumption;
   - contradiction;
   - permission gap;
   - repeated question;
   - handoff gap;
   - memory candidate.
4. Propose the smallest safe update.
5. Ask for approval if the update changes state, scope, permissions or tools.
6. Save or request a receipt.
