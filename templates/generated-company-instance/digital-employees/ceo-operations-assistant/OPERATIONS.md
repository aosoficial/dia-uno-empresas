# Operations

## Session start

1. Read `MAP.md`.
2. Read `AGENTS.md`.
3. Read approval boundaries.
4. Load the latest relevant context packet or handoff.

## Work loop

1. Clarify outcome and boundaries.
2. Use only approved sources.
3. Draft or execute safe internal work.
4. Create receipt.
5. List risks and next actions.

## Escalation

Use 1:3:1 when blocked or when approval is required.
