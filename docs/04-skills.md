# Skills — what agents need to know

These are the core skills a Company Brain agent should have.

They are written as capabilities, not as credits to any external person or private framework.

## 1. Company context reading

The agent can read the Company Brain and identify:

- goals;
- active decisions;
- constraints;
- owners;
- relevant history;
- missing context.

## 2. Context packet creation

The agent can package context before acting.

Output:

- task;
- source facts;
- assumptions;
- missing data;
- approval needs;
- expected output.

## 3. 1:3:1 decision support

When a decision is needed, the agent presents:

- 1 problem;
- 3 possible solutions;
- 1 recommendation.

The agent must not hide uncertainty behind a confident answer.

## 4. Permission-aware execution

The agent knows what it can do alone and what requires approval.

It stops when actions involve:

- customers;
- public content;
- money;
- legal commitments;
- private data;
- production systems.

## 5. Receipt writing

The agent leaves evidence after meaningful work.

A receipt must include:

- action;
- context used;
- files/records changed;
- status;
- outcome;
- next step.

## 6. Handoff

The agent can transfer work to another human or agent without losing context.

A good handoff includes:

- current state;
- owner;
- blocker;
- evidence;
- decision needed;
- deadline.

## 7. Method improvement

The agent can detect repeated failure and propose improvements to:

- templates;
- skills;
- permissions;
- review cadence;
- onboarding;
- tool setup.

## 8. Privacy discipline

The agent knows how to classify data and avoid leaking sensitive material.

Default rule:

> If data is private, real, customer-specific, financial, legal, medical, credential-like, or production-related, stop and ask.

## Recommended first agent skills

Start with these:

1. Company Brain reader.
2. Context packet builder.
3. Decision assistant.
4. Receipt writer.
5. Weekly reviewer.

If your agents cannot perform these reliably, ask for implementation help: https://example.com/libera/company-brain-help
