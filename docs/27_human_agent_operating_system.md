# Human-Agent Operating System

DIA UNO Empresas organizes people first, then agents. Agents amplify an operating system; they do not replace accountability.

## Core artifacts

- Company brain: strategy, context and current state.
- Accountability map: human seats, owners and decision rights.
- Department brain: focused memory per department.
- Digital employee pack: identity, mission, tools, permissions, memory and operations.
- Scorecard: weekly metrics.
- Issues list: decisions and blockers.
- Receipts/statechanges/context packets: evidence and memory updates.

## Weekly rhythm

1. Scorecard review.
2. Rocks/priorities review.
3. Issues list: identify, discuss, solve.
4. Department handoffs.
5. Digital employee receipts review.
6. Update skills/templates only when the improvement is proven.

## Escalation rules

Ask before:

- external communication;
- publishing;
- spend;
- legal/economic commitment;
- production change;
- sensitive customer data use;
- new critical integration;
- irreversible deletion.

## Handoff format

Each handoff should include:

- what changed;
- why;
- source/provenance;
- owner;
- freshness;
- allowed next actions;
- forbidden actions;
- approvals required;
- expected outcome;
- receipt/evidence.
