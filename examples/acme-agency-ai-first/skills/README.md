## Executable skill templates

Copy these public-safe templates into the private company instance before activating a department-specific digital employee:

- `templates/skills/product-discovery-brief.md` — turn a vague product/service idea into a discovery brief.
- `templates/skills/prd-drafting.md` — draft a product requirements document.
- `templates/skills/web-launch-qa.md` — review websites before public/client launch.
- `templates/skills/proposal-drafting.md` — draft proposals from approved commercial context.
- `templates/skills/lead-qualification.md` — score and route opportunities.
- `templates/skills/client-onboarding.md` — convert signed clients into delivery-ready context.
- `templates/skills/delivery-qa.md` — review delivery artifacts before handoff.
- `templates/skills/weekly-review.md` — run the weekly operating review.
- `templates/skills/cashflow-review.md` — review cashflow and payment risks.
- `templates/skills/sop-writing.md` — turn repeated work into SOPs.

## Promotion rule

Create or update a skill only after real work proves the pattern. If a human corrects the same behavior twice, propose a skill/template update and record a statechange.
