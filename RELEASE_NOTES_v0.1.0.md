# Release notes v0.1.0

Company Brain System v0.1.0 introduces the first public-ready version of the framework.

## Included

- Method docs.
- Quick Start.
- Company Brain architecture.
- Operational memory model.
- Agent onboarding guide.
- Agent Runtime Pack templates.
- Department Brain templates.
- Scorecards.
- YAML schemas.
- Validation scripts.
- Synthetic example.
- Safety evaluation guide.

## Not included

- Real client data.
- Private implementations.
- Credentials or live tool configuration.
- Legal, financial, medical, security or compliance advice.
