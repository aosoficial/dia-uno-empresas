# pilares/ — La anatomía (los 5)

Cada función vive en un pilar. Cada pilar trae: qué hace · qué vive en él · cómo se conecta · su
**caja de herramientas con las dos formas (humana y agente)**.

| Pilar | Qué hace |
|---|---|
| [1-rumbo](1-rumbo/) | Decir a dónde va la empresa y qué la gobierna. **Manda.** |
| [2-equipo](2-equipo/) | Que cada función tenga dueño — persona o agente (organigrama híbrido). |
| [3-procesos](3-procesos/) | Que el trabajo se haga igual sin depender de quién esté. |
| [4-caja](4-caja/) | Que la empresa no se quede sin dinero. |
| [5-ejecucion](5-ejecucion/) | Convertir la dirección en movimiento (tracción). |
