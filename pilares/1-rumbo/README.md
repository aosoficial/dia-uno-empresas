# Pilar 1 · Rumbo  *(manda; es la fuente)*

**Qué hace:** decir a dónde va la empresa y qué la gobierna. Todo lo demás existe para servir esto.
**Qué vive aquí:** destino, objetivos (largo/medio/corto), fronteras (lo que no se hará), valores. Los KPIs de todos los pilares nacen aquí.
**Herramienta:** El Rumbo (V/TO). Fijarlo = **humano**; vigilar la deriva y preparar la revisión = **agente**.
